# Preview.ps1 Functional Requirements

**Reminder:** If any new permanent functionality is added to Preview.ps1, update this requirements file to reflect the changes.

- Runs **'npm run preview'** from the project root in a new PowerShell window. **npm is required. Do not use pnpm or yarn.**
- Opens the default browser to http://localhost:4173/ after starting the preview server.
- Informs the user to close the window to stop the server.
